About Page

This app shows the basic usage of Flask framework. It allows creation and deletion of users. A user can self register but will require and approval from administrator before the user can have access to database entries.

Once the user has access, (s)he can create a post and enter expenses. This functionality demonstrates the usage of wtforms. More specifically, usage of dropdown and display of date field and checkboxes.

Further activities to explore:

  • Create a grid editable form
  • Creation of subforms
  • Creation of forms in modals
